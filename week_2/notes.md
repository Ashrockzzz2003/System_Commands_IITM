# wc

```bash
wc filename
```
- lines words bytes filename

```bash
wc -l filename
```
- only first 5 lines

# alias

```bash
alias ll='ls -l'
```
- `ll` will now work the same way as `ls -l`.

```bash
alias
```
- know your aliases

```bash
unalias ll
```
- remove the alias `ll`
